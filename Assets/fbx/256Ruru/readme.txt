# モデル仕様
 - ポリゴン：256
 - テクスチャ：256x256 x1
 - マテリアル：1
 - ブレンドシェイプ：33
 - 対応Mayaバージョン：Maya2023


# リグの使用方法
 1. プロジェクトフォルダを任意の場所に置く。
 2. Mayaを起動し [File > Set Project] から 1 のディレクトリを設定する。(※1)
 3. [File > Create Reference] でシーンデータをリファレンスする。(※2)

 (※1)テクスチャパスを手動で設定する場合は不要です。
 (※2)リグデータ破損を防ぐ為にリファレンスシーンでのモーション作業をおすすめします。

 その他、細かい解説は割愛します。


# トラブルシューティング
 Q. テクスチャの色がおかしい！
 A. ビューポートの [View Transform] を [Un-tpne-mapped(sRGB)] に設定してください。

 Q. 顔のポリゴンがチラつく！
 A. [カメラ(perspShape) > Near Clip Plane] の値を大きくしてください。
